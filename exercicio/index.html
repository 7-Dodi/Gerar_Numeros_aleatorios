Questão 1

Crie um novo projeto usando o template static do Code Sandbox e crie uma página que tenha 1 botão com o rótulo "Gerar". Para essa questão, crie um ouvinte de evento no referido botão que realiza uma chamada para uma função gerarNumero( ) que retorna um número aleatório usando a função Math.random( ). Mostre todos os números gerados, para o usuário em elementos 'li' dentro de um 'ul', mostrando o número mais recente primeiro. (sem CSS).
